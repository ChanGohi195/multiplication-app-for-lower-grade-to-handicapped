
  # 九九練習アプリUI設計

  This is a code bundle for 九九練習アプリUI設計. The original project is available at https://www.figma.com/design/63s2drnuaWcoQCL6dg60RJ/%E4%B9%9D%E4%B9%9D%E7%B7%B4%E7%BF%92%E3%82%A2%E3%83%97%E3%83%AAUI%E8%A8%AD%E8%A8%88.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Environment

  Create a `.env` based on `.env.example` and set:

  - `VITE_SUPABASE_PROJECT_ID`
  - `VITE_SUPABASE_ANON_KEY`
  - `VITE_ADMIN_PASSWORD`

  These values are required at build time but should never be committed to the repository.
